Funktionen

Dieses Addon zeigt Ihnen Ihre Konferenzen direkt in LernSax auf der Startseite an! Für Lehrer werden die Konferenzen der jeweiligen Klasse direkt im Fachlehrerteam angezeigt.

Schritt 1: Addon installieren

Als Erstes musst du das Addon für deinen Browser installieren. Dies ist eigentlich selbsterklärend, du musst nur dem Link zu dem Installer folgen:
Opera Firefox
Die Versionen für Chrome und Edge werden zurzeit von Google und Microsoft überprüft, und werden schnellstmöglich nachgetragen (vorrausgesetzt, sie werden nicht abgelehnt). Falls diese Version abgelehnt werden sollte, oder der Überprüfungsprozess sich zu lange hinziehen sollte, werden wir hier außerdem eine Anleitung hinzufügen, wie Sie das Addon manuell installieren können.

Für Safari wird es allerdings keinen Support geben, da diese zum Veröffentlichen eine Gebühr von ca. 95€/Jahr verlangen.

Schritt 2: Addon anzeigen lassen

Dieser Schritt ist nur für Chrome & Opera notwendig.
Opera Bei Opera ist oben rechts, neben der Adressbar eine Reihe von Zeichen:
Wenn du das Markierte Symbol anklickst, sollte sich dieses Fenster öffnen:
Nun sollte das Addon auch neben der Adressbar angezeigt werden.
Chrome Bei Chrome ist der Vorgang prinzipiell der selbe Vorgang wie bei Opera, nur das Zeichen für Addons ist ein anderes.

Schritt 3: einen Account hinzufügen

Jetzt, wo du das Addon schön sichtbar in deinem Browser hast, musst du nur noch einen Account hinzufügen, damit dir deine Konferenzen in LernSax angezeigt werden:
Lehrer
  • Klicken Sie auf das Addon-Symbol
  • Klicken Sie auf "Klassen & Lehrer-Accounts"
  • Wählen Sie "Hinzufügen" und dann "Lehrer"
  • Geben Sie nun bei LernSax-Namen Ihre LernSax-Email-Adresse ein, sowie bei Klassen-ID Ihre Lehrerkennung auf planungstool-fsg.de
    • Schüler
    1. Klicke auf das Addon-Symbol
    2. Klicke auf "Klassen & Lehrer-Accounts"
    3. Wähle "Hinzufügen". Es sollte "Schüler" ausgewählt sein.
    4. Gebe nun bei Klassen-ID die ID deiner Klasse auf planungstool-fsg.de ein. Diese ID sind die letzten 4 Buchstaben des Links, der zu deinen Konferenzen führt:

      https://planungstool-fsg.de/id/abcd

      Alternativ kannst du auch einfach direkt den Link eingeben.

    Wenn Sie alles richtig gemacht haben, sollten Ihnen nun auf LernSax Ihre Konferenzen angezeigt werden, direkt auf der Startseite, bzw. für Lehrer im Fachlehrerteam jeder Klasse.

    Hinweis für Lehrer*innen Manchmal kann es sein, dass es Ihnen eine Fehlermeldung anzeigt, in der sie aufgefordert werden, die Klassen-IDs neu zu Laden. Hier finden sie dazu nochmal eine Anleitung:
    1. Klicken Sie auf das Addon-Symbol
    2. Klicken Sie auf Accounts
    3. Wählen Sie Ihren Account aus. Es sollte ein Button angezeigt werden mit der Aufschrift "Klassen-IDs erneut laden"
    4. Klicken Sie diesen Button

      5. Damit sollten alle Klassen-IDs auf Ihrem PC wieder aktuell sein.
